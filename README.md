# Tank Brawl

A 3 vs 3 2D top-down multiplayer tank combat game built with Unity. 

## 🎮 Game Description

Tank Brawl is a fast-paced multiplayer game where players control tanks in intense top-down combat. Battle against other players in real-time as you try to get point capturing the zone, dodge incoming fire, and eliminate your opponents.

## ✨ Features

- **Multiplayer Combat** - Real-time multiplayer gameplay using Unity Netcode (Host Client-Autorithative)
- **Top-Down Perspective** - 2D top-down view
- **Tank Battles** - Control powerful tanks with unique weapons
- **Session-Based Gameplay** - Join and create game sessions with other players

## 🛠️ Built With

- **Unity** - Game engine
- **Unity Netcode for GameObjects** - Multiplayer networking
- **C#** - Game programming

## ⚠️ Important Notice

This repository contains **only the game logic** (scripts and core systems) of Tank Brawl. Art assets, animations, prefabs, and other resources are excluded from the repository. As a result, **the project cannot be directly built or run from source**.

1. Clone the repository:
```bash

```

2. Open the project in Unity Hub (Unity 2022.3 or later required)

3. Note: The project will be missing assets and **cannot be played** directly from the editor without the full asset set.

## 🎯 How to Play

1. Launch the game
2. Create or join a multiplayer session
3. Control your tank using WASD/Arrow keys
4. Aim and shoot at enemy tanks
5. Survive and eliminate opponents to win
